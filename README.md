
Customer Churn: Prediction & Retention Strategy

Project Overview
This project aims to analyze customer behavior in the telecommunications sector and predict churn. By developing a Machine Learning model (Random Forest), we identify key factors that drive customers to leave, allowing the company to implement proactive retention strategies. This project demonstrates the end-to-end data science lifecycle, from data cleaning to predictive modeling and strategic recommendations.

Key Features

* Exploratory Data Analysis (EDA): Uncovering patterns in customer tenure, monthly charges, and contract types.
* Predictive Modeling: Utilizing a Random Forest Classifier to detect at-risk customers.
* Strategic Insights: Translating model predictions into actionable business recommendations to improve customer loyalty.
* Interactive Visualizations: Using Plotly to create dynamic charts that reveal data insights.

Model Performance

* Model: Random Forest Classifier
* Primary Metric: Recall (Optimized to 77%) to ensure high coverage of at-risk customers.
* Threshold Tuning: Adjusted to 0.3 to prioritize capturing potential churners over raw accuracy.

Tech Stack

* Language: Python
* Data Processing: Pandas, NumPy
* Visualization: Plotly, Seaborn
* Modeling: Scikit-learn (Random Forest)
* Environment: Kaggle, Jupyter Notebooks


Strategic Recommendations

1. Contract Conversion: Shift month-to-month contract users to long-term plans through targeted loyalty discounts.
2. Early Intervention: Target customers identified by the model with high churn probability (0.3+) for personalized retention offers.
3. Tenure-Based Rewards: Enhance rewards for long-term customers to increase their Lifetime Value (LTV).

Repository Structure

* customer-churn-prediction-retention.ipynb: The core Jupyter Notebook containing the full analysis and model development.
